Tecnologías y Diseño
-React + Vite: Para la estructura de la aplicación.
-TailwindCSS: Para los estilos y diseño responsivo.
-Fetch API: Para la conexión con el Backend.

Configuración e Instalación
Abrir la terminal en la carpeta `front-end-auth` y ejecutar:
terminal bash:
npm install

Conectar api con la front:
Crear un archivo `.env` en la raíz del proyecto y Pega:
VITE_API_BASE_URL=http://localhost:3000/

Correr el proyecto:
npm run dev
